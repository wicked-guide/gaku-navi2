# gaku-navi2

git でクローンしたら

### まずは setup

```
yarn install
```

### 基本は vue cli で管理

```
vue ui
```

ui 上で基本的な操作が可能

- サーバー起動
- アプリビルド
- github-page へのデプロイ

### ライブラリのアップデートはコマンド

```
yarn upgrade
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
